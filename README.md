# 🎧 Аудиогид по КМВ

Интерактивный аудиогид по курортному парку Ессентуков с интеграцией в Telegram.

## 📱 Возможности

- Интерактивная карта с маршрутом
- Аудиоэкскурсии по достопримечательностям
- Автоматическое воспроизведение по GPS
- Офлайн-режим
- Умный плеер с регулировкой скорости
- Адаптивный дизайн для мобильных устройств
- Админ-панель для управления контентом
